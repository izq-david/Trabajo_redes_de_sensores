# Proyecto: Monitorización Biomecánica en Ciclismo mediante IMUs

## Especificación del Proyecto
Este repositorio contiene el software, análisis de datos y la propuesta de hardware para evaluar la viabilidad de las Unidades de Medición Inercial (IMUs) en la monitorización de la cinemática articular y la estimación de potencia en ciclismo. El objetivo es desarrollar una solución de bajo coste aplicable en entornos ecológicos (Ruta y MTB) que iguale o se aproxime a la precisión de los sistemas de laboratorio, implementando algoritmos para la reducción de ruido y deriva temporal (drift).
